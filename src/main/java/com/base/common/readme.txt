|_annotation：放置项目自定义注解
|_aspect：放置切面代码
|_config：放置配置类
|_constant：放置常量、枚举等定义
   |__consist：放置常量定义
   |__enums：放置枚举定义
|_exception：放置自定义异常类
|_handler：放置自定义处理类
|_controller：放置控制器代码
|_filter：放置一些过滤、拦截相关的代码
|_mapper：放置数据访问层代码接口
|_model：放置数据模型代码
   |__bo：放置业务实体对象定义
   |__dto：放置数据传输对象定义
   |__entity：放置数据库实体对象定义
   |__vo：放置显示层对象定义
|_service：放置具体的业务逻辑代码
   |__intf：放置业务逻辑接口定义
   |__impl：放置业务逻辑实际实现
|_sql：放置建表语句
|_utils：放置工具类和辅助代码
